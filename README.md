# AI & Agentic AI for Food Quality Monitoring and Cold Storage Optimization

An end-to-end intelligent system that combines **Computer Vision, IoT sensor analytics, Machine Learning, and Agentic AI** to monitor food freshness and optimize cold storage.

## 🚀 Features

* 📦 Upload food images + sensor data
* 🧿 YOLOv8 object detection for food identification
* 🌡️ Sensor-based spoilage prediction
* 🤖 ML models for freshness estimation
* 📊 Automated freshness report generation
* 📉 Visualization dashboards
* ⚠️ Spoilage alert system
* 📧 Email notifications
* 💬 Interactive AI food storage assistant (Agentic AI)

---

## 🏗️ System Pipeline

1. Upload food images (ZIP)
2. Upload sensor data (CSV)
3. Detect food items using YOLOv8
4. Match items with sensor readings
5. Predict freshness & time-to-expire
6. Generate report
7. Visualize results
8. Agent scans for spoilage and alerts users
